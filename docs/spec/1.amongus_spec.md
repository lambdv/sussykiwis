Vertical Slice Spec: Among Us-like Multiplayer Game Loop
Goal

Build a minimal complete multiplayer loop for a 10-player social deduction game:

10 players total
2 imposters
Crew mates complete tasks, report bodies, vote
Imposters kill, sabotage, and win by parity
Sheriff is a crew role with a kill button
Dead players become ghosts
Game ends in a win scene
1. Core Game States
type GamePhase =
  | "lobby"
  | "playing"
  | "task_interaction"
  | "meeting"
  | "ejection"
  | "win";
Phase Flow
Lobby
  ↓
Playing
  ↓
Meeting / Task Interaction / Win
  ↓
Ejection
  ↓
Playing or Win

Important rule:

Tasks do not change scene. They are an interaction state inside the normal game scene.

type PlayerRole =
  | "crewmate"

  | "alive"
  | "dead"
  | "ghost";

  role: PlayerRole;
  lifeState: PlayerLifeState;

  position: Vec2;
  color: string;

  canMove: boolean;

  killCooldownEndsAt: number;

Role Rules
Role	Team	Can Kill	Can Do Tasks	Can Sabotage
Crewmate	Crew	No	Yes	No
Sheriff	Crew	Yes	Yes	No
Imposter	Imposter	Yes	Fake or No	Yes

For the vertical slice, sheriff can kill any alive player. You can add wrong-kill punishment later.

3. Win Conditions

Run this after:

A kill
A vote/ejection
A disconnect
A task completion
function checkWin(players: Player[]): WinState | null {
  const aliveCrew = players.filter(
    p => p.lifeState === "alive" && p.role !== "imposter"
  ).length;

  const aliveImposters = players.filter(
    p => p.lifeState === "alive" && p.role === "imposter"
  ).length;

  if (aliveImposters === 0) {
    return { winner: "crew", reason: "all_imposters_ejected" };
  }

  if (aliveImposters >= aliveCrew) {
    return { winner: "imposters", reason: "imposter_parity" };
  }

  return null;
}

Optional task win:

if (allCrewTasksComplete) {
  return { winner: "crew", reason: "tasks_complete" };
}

6. Kill System
Imposter Kill
type KillRequest = {
  killerId: string;
  targetId: string;
};

Server validates:

killer.role === "imposter"
killer.lifeState === "alive"
target.lifeState === "alive"
distance(killer, target) <= KILL_RANGE
now >= killer.killCooldownEndsAt
game.phase === "playing"

On success:

target.lifeState = "dead";
spawnDeadBody(target.position, target.id);
killer.killCooldownEndsAt = now + 30_000;
checkWin();
Sheriff Kill

Same cooldown rule:

now >= sheriff.sheriffKillCooldownEndsAt

For vertical slice:

sheriff can kill any alive target

Later improvement:

if sheriff kills crewmate:
  sheriff dies too

type VoteTarget = string | "skip";

type Vote = {
  voterId: string;
  target: VoteTarget;
};

Server validates:

voter.lifeState === "alive"
game.phase === "meeting"
voter has not already voted
Vote Resolution
function resolveVotes(votes: Vote[]): VoteTarget | null {
  // highest vote wins
  // tie = no ejection
}

Result cases:

Player ejected
No one ejected because of tie
No one ejected because skip won

After ejection:

ejectedPlayer.lifeState = "ghost";
checkWin();
game.phase = "ejection";

Then after ejection scene timer:

game.phase = "playing";
9. Ghost System

When a player dies or is ejected:

player.lifeState = "ghost";
player.canMove = true;

Ghost rules:

Can move around world
Can pass through players/doors/walls if you want
Cannot report bodies
Cannot vote
Cannot kill
Cannot sabotage
Can interact only with ghost-compatible tasks
function canInteract(player, object) {
  if (player.lifeState === "ghost") {
    return object.kind === "ghost_task";
  }

  return object.kind !== "ghost_task";
}

11. Sabotage System

Only imposters can open sabotage menu.

type SabotageKind =
  | "lights_off"
  | "gray_players";
type ActiveSabotage = {
  kind: SabotageKind;
  startedAt: number;
  endsAt: number;
};
Sabotage 1: Lights Off

Effect:

Reduce crewmate vision for 10 seconds.
activeSabotages.push({
  kind: "lights_off",
  startedAt: now,
  endsAt: now + 10_000,
});
Sabotage 2: Gray Players

Effect:

All player characters become gray for 10 seconds.
activeSabotages.push({
  kind: "gray_players",
  startedAt: now,
  endsAt: now + 10_000,
});

Client rendering rule:

if (isSabotageActive("gray_players")) {
  renderPlayerColor("gray");
} else {
  renderPlayerColor(player.color);
}

13. World Snapshot
type WorldSnapshot = {
  serverTime: number;
  phase: GamePhase;

  players: SnapshotPlayer[];
  deadBodies: DeadBody[];
  interactables: Interactable[];
  activeSabotages: ActiveSabotage[];
};
type SnapshotPlayer = {
  id: string;
  name: string;
  x: number;
  y: number;
  color: string;
  lifeState: PlayerLifeState;

  // Only visible to self, or only sent privately
  role?: PlayerRole;

  killCooldownRemaining?: number;
};

Important:

Do not reveal everyone’s role to every client. The server should only send role info privately where appropriate.
